# box4399_assistant

#### 介绍
- 自用的4399游戏盒签到库，可以满足简单的自动签到和抢奖品功能。
- 仓库内容比较混乱，稍后会进行整理。

#### 软件架构
Python为主

#### 仓库结构介绍
##### python-mainstream folder
- 包含了可以正常使用的python脚本
- 包含以下功能：
    - 自动获取积分
    - 检测加速卡补仓状态
    - 自动抢加速卡并签到
    - 自动补签+签到当天
    - 自动抢每月奖励

##### 易语言 folder
- 包含了exe格式的脚本文件，支持功能更多
